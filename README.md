# Serverless Order Api

Serverless is a portfolio backend project built with Java 17, AWS Lambda, API Gateway, DynamoDB. 

# Goal

To achieve knowledge and practice during my "AWS Serverless" course in Cloud Development program.

# Tech Stack 

### Programing Language & Utility
- Java 17
- Maven
- IntelliJ IDEA
- Git 
- GitHub

### AWS
- AWS Lambda
- API Gateway
- DynamoDB
- CloudWatch


# Project 

**Order API**

System will be able to:
- create one order
- get an order by Id
- get list of orders from a customer
- delete an order

Small enough or big enough project to practice AWS lambda functions.

# Architecture

The system consists of 3 main parts:

**API Gateway** - handels HTTP.
**Lambda** - executes Java-code.
**DynamoDB** - stores data.
